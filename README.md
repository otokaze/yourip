# yourip
*find your public ip & hostname with docker.*

And, this may also be a good Drone CI demo.

## Quick start 
### Install
```shell
go get github.com/otokaze/yourip
```
### Starting
```shell
yourip --http // use -h show help message.
```
### OR Docker
```shell
docker run --rm -p 80:80 otokaze/yourip
```
